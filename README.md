# Feane Restaurant Management System 🍽️

**Feane Restaurant Management System**, restoranın rəqəmsal idarəetməsini sadələşdirmək üçün hazırlanmış, ASP.NET Core MVC arxitekturası ilə qurulmuş tam funksional Admin Paneli layihəsidir. 

Layihə restoranın menyusunu, rezervasiyalarını və müştəri əlaqələrini dinamik şəkildə idarə etməyə imkan verir.

## 🚀 Əsas Funksionallıqlar
* **Dinamik İdarəetmə Paneli:** Restoranın cari statusunu, aktiv yeməkləri və rezervasiyaları real vaxt rejimində izləmə.
* **CRUD Əməliyyatları:**
    * **Slider:** Ana səhifə bannerlərinin idarə olunması.
    * **Dish (Yeməklər):** Menyunun tam idarə edilməsi.
    * **Discount Product:** Endirimli məhsulların tənzimlənməsi.
    * **BookTable:** Müştəri rezervasiyalarının qəbulu və idarə olunması.
* **Modern UI/UX:** Neon detallar və glassmorphism effekti ilə hazırlanmış qaranlıq (Dark Mode) mövzu.
* **Avtomatlaşdırılmış Panellər:** Dinamik salamlamalar, saat və şefin günün sözü funksiyaları.

## 🛠️ Texnoloji Yığın (Tech Stack)
* **Framework:** ASP.NET Core MVC
* **Language:** C#
* **Database:** Entity Framework Core (SQL Server)
* **Frontend:** HTML5, CSS3 (Bootstrap 5), jQuery, Font Awesome 6
* **Fonts:** Google Fonts (Playfair Display, Roboto, Open Sans)
