# Feane Restaurant Management System 🍽️

**Feane Restaurant Management System**, restoranın rəqəmsal idarəetməsini sadələşdirmək üçün hazırlanmış, ASP.NET Core MVC arxitekturası və **PostgreSQL** verilənlər bazası ilə qurulmuş tam funksional Admin Paneli layihəsidir. 

## 🚀 Əsas Funksionallıqlar
* **Dinamik İdarəetmə Paneli:** Restoranın cari statusunu, aktiv yeməkləri və rezervasiyaları real vaxt rejimində izləmə.
* **CRUD Əməliyyatları:** Slider, Dish (Yeməklər), Discount Product,BookTable,Customers üçün tam idarəetmə.
* **Modern Admin Panel UI/UX:** Neon detallar və glassmorphism effekti ilə hazırlanmış qaranlıq (Dark Mode) mövzu.
* **Avtomatlaşdırılmış Panellər:** Dinamik salamlamalar, saat və şefin günün sözü funksiyaları.

## 🛠️ Texnoloji Yığın (Tech Stack)
* **Framework:** ASP.NET Core MVC
* **Language:** C#
* **Database:** **PostgreSQL** (Entity Framework Core vasitəsilə)
* **Frontend:** HTML5, CSS3 (Bootstrap 5), jQuery, Font Awesome 6
